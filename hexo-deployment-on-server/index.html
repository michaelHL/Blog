这里的 root 即为网站所在目录，大量的教程中将 /var/www/ 设成网站所在目录，估计是约定俗成吧。那么这里就有权限问题了，刚入坑的我还并不是很了解 Linux 的权限机制，所以干脆决定在普通用户下生成静态文件，并同步至 Github，再利用系统用户利用 Webhook 从 Github 更新网站目录。
